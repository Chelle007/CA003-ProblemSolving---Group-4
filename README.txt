# EchoFlowerApp

## Description

EchoFlowerApp is a group project developed for Problem Solving CA003. It is a flower shop application built with Python, featuring a graphical user interface (GUI) and utilizing SQLite for data storage.

(Note: the UI is not responsive)

## Prerequisites

Before running EchoFlowerApp.py, make sure you have the following prerequisites installed and configured:

1. **Display Settings:**
   - Set the scale to 125% and display resolution to 1920 x 1080 on your system settings.
     - **Windows:**
       - Go to Settings > System > Display > Scale & Layout.
       - Change the scale to 125% and display resolution to 1920 x 1080.
     - **Mac OS:**
       - Go to System Preferences > Displays.
       - Choose the 'Default for display' option, usually 1920 x 1080.
   - Note: The fixed design size may cause differences in widget and text sizes on different scales and resolutions.

2. **Install pip (if not installed yet):**
   - Follow the [official installation guide](https://pip.pypa.io/en/stable/installation/) to install pip.

3. **Install customtkinter:**
   - Open Command Prompt and type `pip install customtkinter`.

4. **Install packaging:**
   - Open Command Prompt and type `pip install packaging`.

5. **Install pillow:**
   - Open Command Prompt and type `pip install pillow`.

6. **Install TT Commons Medium (Font):**
   - Open `TTCommons Medium.otf` and press 'Install'.

## Adding Blooms and Add-ons to the Database

To add remaining blooms and add-ons to the database:

2. **Navigate to 'Add Item':**
   - Click on 'Add Item' located on the top right corner.
3. **Select 'TXT File' and Add:**
   - Choose 'TXT File' and click 'Add' once (no need to enter the file name).
4. **Change Type to 'Bloom':**
   - After adding the TXT file, change the type to 'bloom'.
5. **Save Changes:**
   - Click 'Add' once again (no need to enter the file name).
